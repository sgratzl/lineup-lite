## Install LineUp-Lite

```sh
npm install @lineup-lite/table
```

## Define LineUp-Lite data structure

```js
const rows = Array(100).fill(0).map((, i) => ({
  name: `Person ${i}`, 
  age: Math.floor(Math.random() * 80), 
  status: ['single', 'married', 'separated'][Math.floor(Math.random() * 3) % 3], 
}));
```
