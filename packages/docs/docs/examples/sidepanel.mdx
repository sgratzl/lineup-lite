LineUp-lite supports showing a [Side Panel](/docs/table/sidepanel).

The side panel can be rendered next to LineUp-lite or whereever you want but it has to under the same shared `LineUpLiteStateContextProvider`,
which is used to share the react-table instance and state between the table and side panel.
