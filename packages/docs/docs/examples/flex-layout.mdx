LineUp-lite supports different react-table layouts. The default layout (`featureDefaultLayout`) is an alias for `useBlockLayout`.
In addition, there is flex layout (`featureFlexLayout` aka `useFlexLayout`) which uses CSS flex layout to stretch the table automatically to its width.
