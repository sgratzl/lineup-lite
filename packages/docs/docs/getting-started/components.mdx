The LineUp-lite components are a set of basic React visualization components and methods for computing statistics.

LineUp-lite supports multiple different attribute types:

- [text](/docs/components/text)
- [categorical](/docs/components/categorical)
- [number](/docs/components/number)
- [date](/docs/components/date)
- [set](/docs/components/set)

## Install LineUp-lite Components

```sh
npm install @lineup-lite/components
```

or

```sh
yarn add @lineup-lite/components
```

## Example

```jsx live
// import { TextLabel, NumberBar, CategoricalColor, DateLabel } from '@lineup-lite/components';

function Example() {
  const row = {
    name: 'Panchito Green',
    age: 10,
    shirtSize: 'S',
    birthday: new Date(2011, 1, 1),
  };

  return (
    <div style={{ maxWidth: 200 }}>
      <TextLabel value={row.name} />
      <NumberBar value={row.age} scale={normalize(0, 50)} />
      <CategoricalColor value={row.shirtSize} color="red" />
      <DateLabel value={row.birthday} />
    </div>
  );
}
```
